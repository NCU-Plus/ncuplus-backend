# NCU Plus Backend

## Description

NCU Plus website backend.

## Installation

```bash
$ docker-compose up -d --build
$ docker exec -it ncuplus-backend_app_1 bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
